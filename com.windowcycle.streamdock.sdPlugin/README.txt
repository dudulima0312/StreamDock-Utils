====================================
  WINDOWCYCLE PLUGIN FOR STREAMDOCK
====================================

Version: 1.0.0
Author: MiraBox Community

DESCRIPTION:
Navigate through open windows using Task View (Alt+Tab) with knob rotation.

INSTALLATION:
1. Drag this folder into StreamDock application
   OR
2. Copy this folder to: %appdata%\Elgato\StreamDeck\Plugins\
3. Restart StreamDock

USAGE:
- Rotate Knob: Opens Task View and navigates between windows
- Stop Rotating: Auto-selects after 1.5 seconds
- Press Knob: Immediately confirms selection

REQUIREMENTS:
- Windows 7 or later
- StreamDock version 3.10.188.226 or later
- Administrator permissions recommended

FEATURES:
- Native Alt+Tab simulation
- Automatic Alt key management
- Smooth knob navigation
- Customizable sensitivity and delay
- Multi-language support

CONFIGURATION:
Click the action in StreamDock to configure:
- Rotation Sensitivity (1-5 ticks)
- Auto-release Delay (0.5-5 seconds)
- Reverse Direction
- Knob Press behavior

TROUBLESHOOTING:
- If not working, run StreamDock as Administrator
- Check Windows Defender exceptions
- View logs at: %appdata%\Elgato\StreamDeck\logs\
